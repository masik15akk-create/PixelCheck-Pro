🧠 Концепция
PixelCheck Pro — это утилита для проверки экрана на битые и залипшие пиксели. Она позволяет:

✅ Заполнять экран однородными цветами (чёрный, белый, красный, зелёный, синий, серый, жёлтый, пурпурный, голубой).

✅ Переключать цвета по клику мыши, нажатию пробела или автоматически.

✅ Полноэкранный режим без рамок и панелей — максимальное покрытие дисплея.

✅ Отображение информации о текущем цвете и номере режима.

✅ Горячие клавиши для быстрого управления (пробел — следующий цвет, ESC — выход).

✅ Поддержка нескольких мониторов — можно переключаться между дисплеями (в некоторых версиях).

✅ Режим «сканирования» — плавное изменение цветов для выявления дефектов.

✅ Увеличение — приближение области экрана для детального осмотра (в некоторых версиях).

🚀 Как запустить
Каждая версия использует соответствующую библиотеку для создания оконного приложения. Установите необходимые зависимости и запустите.

Python
bash
pip install tkinter  # обычно уже установлена
python pixelcheck_python.py
C++
bash
# Требуется SFML (sudo apt install libsfml-dev)
g++ -std=c++17 pixelcheck_cpp.cpp -o pixelcheck -lsfml-graphics -lsfml-window -lsfml-system
./pixelcheck
Java
bash
javac PixelCheckJava.java && java PixelCheckJava
C# (.NET Core)
bash
# Windows Forms доступна в .NET Core
dotnet add package System.Windows.Forms
dotnet run
Go
bash
# Требуется fyne (go get fyne.io/fyne/v2)
go mod tidy
go run pixelcheck_go.go
Rust
bash
# Требуется minifb (cargo add minifb)
cargo run
JavaScript (браузер)
bash
# Просто откройте pixelcheck_js.html в современном браузере
# Для полноэкранного режима нажмите F11
